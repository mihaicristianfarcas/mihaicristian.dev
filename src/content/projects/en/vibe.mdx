Vibe is my own go at the "chat with an AI, get an app" idea. You describe what
you want, it generates a working application, and you watch it run in a live
preview beside the conversation.

Generation turned out to be the easy half. Running code an AI has just written
is a genuine security problem, so every project gets its own isolated cloud
sandbox and nothing generated ever touches the host. That constraint ended up
shaping the whole architecture.

Nobody paid me for this one. It was the most fun I'd had with an uncomfortable
engineering problem in a while.
