*Ukiyo*, the floating world. A modal terminal text editor written in Rust,
built to answer a question I'd been carrying around for years: what's actually
in there?

I'd used editors every day for years without once thinking about the machinery.
How does a buffer really work? What is a mode, mechanically? Why is undo so much
harder than it looks? The only way I know to stop wondering about something is
to build it, so I did — modes, a file explorer, visual line wrapping, and a
command-based undo system. Every one of those I understood properly only after
implementing it.

It's also where I got to use the State, Command, and Observer patterns somewhere
they genuinely fit, instead of where a textbook wanted them. That taught me more
than the editor did.
