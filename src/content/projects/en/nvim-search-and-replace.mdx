Search and replace across a whole project has never had tooling I fully trust.
You either read a quickfix list like a compiler dump, or you fire a `sed`
command blind and hope.

So this one shows you the change before you make it. Results stream in from
ripgrep as they're found, every match gets a side-by-side before and after, you
tick the ones you want, and the whole thing is undoable afterwards anyway.
Before it writes, it checks the exact text still matches, so a stale result
can't quietly corrupt a file.

I built it because I wanted it, and I use it every day. Other people found it
and use it too.
